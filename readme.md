## Local Testing
Open index.html in your web browser to preview the site.

## Customization (IMPORTANT)

### 1. Update Your Contact Info
Edit all HTML files and replace:
- Email: info@alphaclasses.com ? YOUR EMAIL
- Phone: +91 98765 43210 ? YOUR PHONE
- Address: 123, School Road ? YOUR ADDRESS

### 2. Setup Google Form (for enrollment)
1. Go to forms.google.com
2. Create form with fields: Student Name, Parent Name, Email, Phone, Class, Subject
3. Get the embed code and replace in contact.html

### 3. Setup Gumroad (for selling materials)
1. Create account at gumroad.com
2. Upload your study materials
3. Get product links and update materials.html

## Deploy to GitHub Pages
1. Push to GitHub
2. Go to repository Settings ? Pages
3. Enable GitHub Pages
4. Site goes live at: https://your-username.github.io
